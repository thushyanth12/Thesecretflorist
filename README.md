# The Secret Florist

Welcome to **The Secret Florist**, a beautifully crafted e-commerce platform dedicated to handcrafted bouquets for life's most beautiful moments. Designed with a premium "Blush Pink" aesthetic, the site offers a seamless shopping experience for fresh flowers with a unique WhatsApp-integrated checkout and customization process.

## Features

- **Elegant UI & "Blush Pink" Aesthetic**: A modern, visually stunning interface using soft pastels, dynamic gradients, glassmorphism elements, and micro-animations to create a premium feel.
- **Dynamic Product Catalog**: Extensive selection of bouquets categorized by occasions and styles (e.g., Birthday, Money Bouquet, Perfume Bouquet, Luxury, Budget-friendly).
- **"Customize Bouquet" Integration**: Allows users to customize their bouquets directly or order specialized items (like Perfume Bouquets) by sending a pre-filled WhatsApp message, complete with product preview capabilities.
- **Shopping Cart**: Fully functional client-side cart allowing users to add, remove, and adjust bouquet quantities before proceeding to checkout.
- **WhatsApp Checkout**: Direct checkout to WhatsApp, sending an order summary directly to the florist for quick, personal service and same-day delivery arrangements.
- **Responsive Design**: Optimized for seamless navigation across both desktop and mobile devices, featuring a sleek desktop navbar and a persistent mobile navigation bar.
- **SEO Optimized & Accessible**: Built with proper semantic HTML, Open Graph meta tags, ARIA attributes, and performance best practices.

## Technology Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3 (Vanilla)**: Custom styling using CSS variables, flexbox/grid layouts, animations, and responsive media queries.
- **JavaScript (Vanilla)**: Handles DOM manipulation, cart logic, category filtering, modal interactions, and WhatsApp API integration.
- **FontAwesome**: For intuitive iconography.
- **Google Fonts**: Utilizing elegant typography ('Cinzel', 'Outfit', 'Cormorant Garamond').

## Setup & Running Locally

1. **Clone or Download the Repository:**
   Ensure you have the project files including `index.html`, `style.css`, `script.js`, and the `assets` folder.

2. **Open in Browser:**
   Since this project uses vanilla HTML/CSS/JS without a build step, you can simply open `index.html` in any modern web browser to view the site.
   
   Alternatively, for a better development experience, use a local server like VS Code's "Live Server" extension:
   - Open the project folder in VS Code.
   - Right-click `index.html` and select **"Open with Live Server"**.
